# Misión Cruzada

Web app móvil para 2 jugadores. El móvil solo reparte misiones secretas y marca la ronda. La partida real se juega fuera de la pantalla.

## Stack

- React
- Vite

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Publicar en Vercel

1. Sube esta carpeta a GitHub.
2. Importa el repo en Vercel.
3. Framework preset: `Vite`.
4. Build command: `npm run build`
5. Output directory: `dist`

## Idea de producto

- 2 jugadores
- 1 móvil
- misiones secretas cruzadas
- apto para niños
- partidas cortas y rejugables
